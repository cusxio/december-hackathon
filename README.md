# December Hackathon

## Get Started

```bash
#  Step 1
$ git clone git@github.com:cusxio/december-hackathon.git

# Step 2
$ npm install
```

## Development

```bash
$ npm run start
```

## Production

```bash
# Bundling and Optmizing Files
$ npm run production

# For Github Pages
$ npm run deploy:gh-pages
```
